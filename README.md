# 🎮 Jogo de Adivinhação Numérica
Um jogo simples e interativo desenvolvido para testar suas habilidades de dedução! O sistema escolhe um número secreto e você tem um limite de tentativas para acertar, recebendo dicas pelo caminho.
# 🚀 Como Funciona o Jogo
 * **O Objetivo:** Adivinhar o número aleatório gerado pelo sistema entre **1 e 100**.
 * **Dicas em Tempo Real:** A cada palpite inserido no campo de entrada, a tela informa se você acertou ou se o número secreto está **próximo ou distante**.
 * **Limite de Tentativas:** Você tem no máximo **10 chances**.
 * **Histórico:** A tela exibe o seu saldo restante de tentativas e a lista com todos os números que você já tentou, ajudando na sua estratégia.
# 🛠️ Tecnologias Utilizadas
O projeto foi construído do zero utilizando as tecnologias fundamentais da web:
 * **HTML5:** Estruturação dos elementos da página (input, botão e exibição de textos).
 * **CSS3:** Estilização e design da interface para torná-la amigável e responsiva.
 * **JavaScript (ES6):** Toda a lógica do jogo (geração do número aleatório, validação do input, controle do contador de tentativas e atualização da tela).
