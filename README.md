# Jogo-de-adivinhe-o-n-mero
Jogo de adivinhação em HTML, CSS e JavaScript: o sistema gera um número secreto de 1 a 100 e o usuário tenta acertar via input. A tela exibe se o palpite está correto e dá dicas se o valor está próximo ou distante. O jogador tem até 10 tentativas, com o saldo restante e o histórico de números já tentados atualizados em tempo real.
